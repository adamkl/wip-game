# Scaffold

## Requirements

- The project must build with `npm run build` producing a `dist/` directory
- `npm run dev` must start a Vite dev server with HMR
- TypeScript strict mode must be enabled
- Kaplay must initialise and render a canvas without errors in the browser
- The `proof` scene must display at least one sprite from the sprite sheet on screen, confirming the full asset pipeline is working
