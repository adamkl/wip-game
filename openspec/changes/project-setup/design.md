## Overview

Standard Vite + TypeScript SPA with Kaplay. The only non-obvious part is the sprite extraction pipeline.

```

│       └── proof.ts       # Temporary: renders sprites on screen to verify loading

```

## Sprite Extraction

The PICO-8 `__gfx__` section is a 128×128 grid of hex digits. Each digit is one pixel, mapping to one of 16 palette colours. Colour index 0 = transparent.

**PICO-8 palette (RGB):**
```
0: (0,0,0)       1: (29,43,83)    2: (126,37,83)   3: (0,135,81)
4: (171,82,54)   5: (95,87,79)    6: (194,195,199) 7: (255,241,232)
8: (255,0,77)    9: (255,163,0)   a: (255,236,39)  b: (0,228,54)
c: (41,173,255)  d: (131,118,156) e: (255,119,168) f: (255,204,170)
```

The script embeds the raw `__gfx__` string, walks it row-by-row, and writes a 128×128 RGBA PNG using `pngjs`. Alpha = 0 for colour 0, 255 otherwise.

## Sprite Atlas Layout

Each sprite is 8×8 px in the sheet. Named regions passed to `loadSpriteAtlas()`:

```
Sprite index → sheet position (x = (idx % 16) * 8, y = floor(idx / 16) * 8)

Characters (row 0, indices 1-12):
  "char1_idle": { x:8,   y:0, w:8, h:8 }
  "char1_run":  { x:16,  y:0, w:8, h:8 }
  "char2_idle": { x:24,  y:0, w:8, h:8 }
  "char3_idle": { x:40,  y:0, w:8, h:8 }
  "char3_run":  { x:48,  y:0, w:8, h:8 }
  "char4_idle": { x:56,  y:0, w:8, h:8 }
  "char4_run":  { x:64,  y:0, w:8, h:8 }
  "char5_idle": { x:88,  y:0, w:8, h:8 }
  "char5_run":  { x:96,  y:0, w:8, h:8 }

Tiles (row 8, indices 128-129):
  "tile_block":  { x:0,  y:64, w:8, h:8 }
  "tile_wall":   { x:8,  y:64, w:8, h:8 }
```

Sprite indices were read directly from `character_sprs` in `wip.p8`.

## Kaplay Init

```typescript
kaplay({
  width: 256,
  height: 256,
  scale: 2,          // 2× pixel-art upscale
  crisp: true,       // no bilinear blur on pixel art
  background: [0, 0, 0],
})
```

Scale and canvas size to be tuned once the level dimensions are known (level-01 change).
